Game.Constants.load_asset(root_path: str) → Dict[str, List[Surface]][source]
Look through the folders/files in the root_path.

From the found folders extract list of files to coresponding key (folder name) If found folders contain other folders repeat the process.

Args:

root_path (str): path to the folder containing assets

Returns:

Dict[str, List[pygame.Surface]]: container for image files belonging to coresponding folder name

Returns:
